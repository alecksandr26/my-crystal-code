## Compile C and Crystal with no optimizations:
Crystal:

```$ crystal build crystal.cr```

C:

```$ gcc c.c -o c```

## Compile C and Crystal with no optimizations:
Crystal:

```$ crystal build crystal.cr --release```

C:

```$ gcc c.c -o c -O3```
